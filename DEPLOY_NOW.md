## 🚀 Один клик деплой

### Шаг 1: Открой Render.com

https://render.com

### Шаг 2: Авторизуйся через GitHub

- Нажми "Sign up" или "Sign in"
- Выбери "Continue with GitHub"
- Авторизуй приложение

### Шаг 3: Создай Blueprint deployment

1. Нажми **"New +" → "Blueprint"**
2. Выбери **"Public Git repository"**
3. Введи: `https://github.com/BooRex/ttr`
4. Нажми **"Connect"**

### Шаг 4: Запусти деплой

1. Выбери ветку: **`main`**
2. Нажми **"Deploy"** 🎯
3. Жди **5-10 минут**

### Шаг 5: Получи ссылки

После успешного деплоя:

```
🌐 Frontend:  https://ttr-client.onrender.com
🔌 Backend:   https://ttr-server.onrender.com
```

## 📊 Что произойдет

### Render автоматически:

✅ Установит зависимости (`npm install`)  
✅ Собрет shared пакет (`npm run build -w packages/shared`)  
✅ Собрет backend (`npm run build -w apps/server`)  
✅ Собрет frontend (`npm run build -w apps/client`)  
✅ Запустит оба сервиса  
✅ Создаст URL для каждого  

### Keep-Alive автоматически:

✅ Фронтенд пингует backend каждые 5 минут  
✅ Сервер никогда не засыпает  
✅ Друзья всегда получают быстрый ответ

## 🆘 Что-то пошло не так?

### Посмотри логи в Render Dashboard

1. Открой сервис
2. Нажми **"Logs"**
3. Ищи красные ошибки

### Частые проблемы:

| Проблема | Решение |
|----------|---------|
| `npm ERR` при build | Может быть старый кеш - попробуй редеплой |
| `Port already in use` | Перезагрузи сервис (restart) |
| Фронтенд не загружается | Проверь, что `dist` папка собрана |
| WebSocket не подключается | F12 → Console в браузере → посмотри ошибки |
