# 🚀 Render.com - Quick Deploy

## Один клик деплой

### Вариант 1: Через GitHub (РЕКОМЕНДУЕТСЯ)

1. **Запушь код на GitHub**
   ```bash
   git push origin main
   ```
   *(Если SSH не работает, используй токен)*

2. **Открой https://render.com**

3. **Нажми "New +" → "Blueprint"**

4. **"Public Git repository"** и введи:
   ```
   https://github.com/YOUR_USERNAME/ticket-to-ride-mvp
   ```

5. **Нажми "Connect" → "Deploy"**

6. ✅ **Готово!** Через 5-10 минут получишь URL

---

## Переменные окружения

Все автоматически устанавливаются через `render.yaml`:

| Переменная | Значение |
|-----------|----------|
| NODE_ENV | production |
| VITE_SERVER_URL | https://ttr-server.onrender.com *(автоматически)* |

## 🆘 Если не работает

### Проверь логи в Render Dashboard:
1. Открой сервис
2. Нажми "Logs"
3. Ищи красные ошибки

### Частые проблемы:

| Проблема | Решение |
|----------|---------|
| `npm ERR` при build | Может быть проблема с зависимостями - коммит еще не дошел до GitHub |
| Фронтенд ищет `/dist/` | Это нормально - render.yaml исправляет |
| WebSocket не подключается | Проверь консоль браузера (F12) |
