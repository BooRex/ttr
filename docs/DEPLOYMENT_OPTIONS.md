# Варианты бесплатного хостинга для Ticket to Ride

## Требования приложения
- ✅ Backend: Node.js + Express
- ✅ Frontend: React (статический сборка)
- ✅ WebSockets для real-time multiplayer
- ❌ БД: не требуется (история в памяти)
- ⚡ Малые нагрузки (1 игра одновременно)

---

## Топ рекомендации

### 🥇 1. **Fly.io** (ЛУЧШИЙ ВАРИАНТ)
**Плюсы:**
- Полностью бесплатный слой для малых приложений
- Отличная поддержка WebSockets
- Быстрая инфраструктура (серверы по всему миру)
- Хороший uptime
- Просто развернуть (flyctl CLI)

**Минусы:**
- Нужна карточка для регистрации
- Может заспать через ~30 дней неактивности

**Для твоего случая:** ⭐⭐⭐⭐⭐ ИДЕАЛЬНО

---

### 🥈 2. **Railway**
**Плюсы:**
- $5 кредит/месяц бесплатно (хватает для малого проекта)
- Очень простой деплой (GitHub интеграция)
- Хорошая поддержка WebSockets
- UI очень friendly

**Минусы:**
- Кредит закончится
- После этого либо платить, либо проект упадет

**Для твоего случая:** ⭐⭐⭐⭐ ХОРОШИЙ ВЫБОР

---

### 🥉 3. **Glitch**
**Плюсы:**
- Полностью бесплатный
- Не спит (но может быть медленным)
- Хороший для тестирования с друзьями
- Встроенный редактор кода

**Минусы:**
- Медленнее чем Railway/Fly.io
- Может быть нестабильным
- Ограничения на память/CPU

**Для твоего случая:** ⭐⭐⭐ ЕСЛИ ХОЧЕШЬ БЫСТРО ЗАРАБОТАТЬ

---

### 4. **DigitalOcean App Platform**
**Плюсы:**
- $12/месяц (можешь использовать промокод на $200-300 кредит для новичков)
- Надежно
- Хорошая документация

**Минусы:**
- Платный (хотя с кредитом может быть бесплатно)
- Нужна карточка

**Для твоего случая:** ⭐⭐⭐⭐ ПО НАДЕЖНОСТИ, НО ДОРОЖЕ

## Рекомендация для твоего проекта

### Сценарий 1: "Хочу просто поиграть с друзьями"
**→ Fly.io** 
- Настройка: 15 минут
- Бесплатно навсегда (если платить $5/месяц за CPU)
- Или **Railway** с бесплатным $5 кредитом

### Сценарий 2: "Хочу максимальной стабильности"
**→ DigitalOcean** 
- При регистрации по реф-ссылке дают кредит
- Достаточно для первых 2-3 месяцев

### Сценарий 3: "Быстро что-то протестировать"
**→ Glitch** или **Railway**
- Деплой за 5 минут
- Сразу можешь пригласить друзей

## Локальная сеть (WiFi)

Для игры **локально по WiFi** без интернета:

```bash
# 1. Запусти backend локально
npm run dev -w apps/server

# 2. Запусти frontend локально
npm run dev -w apps/client

# 3. Узнай IP адрес своего компьютера
ipconfig  # Windows - ищи "IPv4 Address" (обычно 192.168.x.x)

# 4. Друзья подключаются по: http://192.168.x.x:5173
```

**Важно:** 
- Все должны быть в одной WiFi сети
- Может быть блокировано фаерволом
- Лучше отключить VPN

## Инструкции по деплою

### Fly.io

```bash
# 1. Установить flyctl
# https://fly.io/docs/hands-on/install-flyctl/

# 2. Авторизоваться
flyctl auth login

# 3. Создать приложение
flyctl launch

# 4. Задеплоить
flyctl deploy
```

### Railway

```bash
# 1. Связать GitHub репо
# 2. Создать проект в railway.app
# 3. Добавить NODE_ENV=production
# 4. GitHub Actions сделает деплой автоматически
```

### Glitch

```bash
# 1. Скопировать проект как remix
# 2. Отредактировать .env
# 3. Автоматически запустится
```
